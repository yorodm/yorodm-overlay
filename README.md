### `rofi-scripts`

Rofi menus courtesy of [cramermarius/rofi-menus](https://github.com/cramermarius/rofi-menus)
includes:

* `rofi-power`, shows a menu for *logout*,*lock*,*suspend*,*reboot*
* `rofi-shot`, shows a menu for taking screenshots
